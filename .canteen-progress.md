## Commands Summary

### Backend
```bash
cd backend
npm install
npm run build
npm start
npm run seed
```

### Admin Panel
```bash
cd admin
npm install
npm run dev
npm run build
```

### Mobile App
```bash
cd app
npm install
npx expo start
```

## Environment Variables Required

### Backend (.env)
- MONGO_URI
- JWT_SECRET (64 char)
- REFRESH_TOKEN_SECRET (64 char)
- QR_SECRET (64 char)
- RAZORPAY_KEY_ID / RAZORPAY_KEY_SECRET / RAZORPAY_WEBHOOK_SECRET
- EMAIL_HOST / EMAIL_PORT / EMAIL_USER / EMAIL_PASS

### Admin (.env)
- VITE_API_URL

### Mobile (.env)
- EXPO_PUBLIC_API_URL
