## API Endpoints

### Auth
- `POST /api/auth/signup` - Register
- `POST /api/auth/login` - Login
- `POST /api/auth/refresh` - Refresh token
- `POST /api/auth/verify-otp` - Verify email OTP

### Menu
- `GET /api/menu` - Get all items
- `POST /api/menu` - Create item (manager/admin)
- `PUT /api/menu/:id` - Update item
- `DELETE /api/menu/:id` - Soft delete

### Orders
- `POST /api/orders/create` - Create order
- `GET /api/orders/my` - Get my orders
- `GET /api/orders/:id` - Get order details
- `POST /api/orders/:id/fulfill` - Fulfill with QR token

### Payment (Mock)
- `POST /payment/mock/create` - Create payment order
- `POST /payment/mock/verify` - Verify payment
- `GET /payment/mock/status/:orderId` - Check status
- `GET /payment/mock/pay/:orderId` - Web payment page

### Admin
- `GET /api/admin/stats` - Dashboard stats
- `GET /api/admin/orders` - All orders
- `GET /api/admin/users` - All users
- `PUT /api/admin/users/:id/role` - Update role

## Deployment Commands

### Backend (Render)
```bash
cd backend
npm install
npm run build
# Deploy via Render dashboard or git push
```

### Admin (Vercel)
```bash
cd admin
npm install
npm run build
# Deploy via Vercel dashboard or git push
```

### Mobile (APK)
```bash
cd app
npx expo prebuild --platform android
cd android
./gradlew assembleRelease
# APK: app/build/outputs/apk/release/app-release.apk
```
